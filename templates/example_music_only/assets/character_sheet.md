# Blueprint Agent

Generate a character sheet focused on emotional resonance and sonic identity.

**Rules:**
- Emphasize personality, emotional range, and thematic essence
- Include details that translate to musical mood/genre
- No placeholders like "[Age]"
- Plaintext only
- Output in single code block
